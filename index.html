GitHub Pagesの特徴

GitHub Pagesは、GitHubリポジトリから直接ウェブサイトをホストするサービスです。

  • 無料で静的なウェブサイトをホストできます。
  • ブランチを指定してデプロイできます。
  • Jekyllなどの静的サイトジェネレーターと統合可能です。
  • GitHubリポジトリから直接コンテンツを提供します。
  • カスタムドメインの設定も可能です。
